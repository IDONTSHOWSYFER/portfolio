# Interactive Portfolio

This `public` folder contains an interactive portfolio designed to provide a smooth and enjoyable user experience. The project is adapted from the original PHP version using PHPMailer and has been modified to be hosted on GitHub, which is a static hosting platform.

## Technologies Used

- **Mail.js**: For handling email submissions and user interactions.
- **HTML/CSS/JavaScript**: For the structure, styling, and interactivity of the portfolio.

## Adaptations

Since GitHub Pages is a static hosting platform, the project has been adapted accordingly to ensure it works seamlessly in this environment. This includes:

- Modifying dynamic features into a static version.
- Utilizing Mail.js to replace PHPMailer, allowing forms to be submitted directly from the frontend.

## Optimization

All content has been minified to enhance user experience and improve loading times. This ensures that users can navigate the portfolio quickly and efficiently.
